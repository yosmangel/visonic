Plugin para adecuación a RGPD LOPD. Paginas legales, banner de cookies, clausulas legales

== Description ==
clickDatos LOPD plugin realiza las siguientes funciones:
1-Creación de las paginas de Aviso legal, Politica de cookies y Politica de privacidad con los datos que se rellenen en el formulario Lopd para formularios.
2-Inserción de banner de politica de cookies con distintas posibilidades de personalización y colocación
3-Inserción de las clausulas de proteccion de datos en los formularios realizados con el plugin Contact Form de la página web.


== Installation ==

1.Sube la carpeta completa click-datos-lopd al directorio /wp-content/plugins/ o busca en el repositorio de plugin de wordpress por RGPD
2.Activa el plugin desde el menú ‘Plugins’ de WordPress.
3.Si usted tiene instalado cualquier plugin de chache es recomendable borrar cache una vez se ha terminado la instalación y activación
Encontrarás el menú ‘RGPD clickDatos’ en tu panel de administración de WordPress.

== Frequently Asked Questions ==
= ¿Páginas necesarias? =
En este plugin se incorporarán las páginas de Política de Privacidad, Política de Cookies y Aviso Legal, en estas aparecerá toda la información referente a la empresa o entidad que recoge y se hace responsable de los datos recibidos, dónde y cómo el usuario puede dirigirse a la empresa para ejercer sus derechos ARCO.
La RGPD busca así mayor transparencia en Internet para proteger al usuario frente a cualquier tipo de violación de su información de carácter personal.

= ¿Qué cláusulas son necesarias? =
Hay tres tipos de cláusulas de las cuales una es siempre obligatoria y las otras dos opcionales.
•	Cláusula de tratamiento de información: esta cláusula es obligatoria para todos los formularios de su web. La cual comunica al usuario como almacenará la empresa los datos que se registren del mismo y establece quien es el responsable encargado del tratamiento, también informa de cómo podemos ejecutar los derechos ARCO sobre los datos registrados de carácter personal.
•	Cláusula de mayores de 18 años: esta cláusula solo será necesaria en caso de que tu web proporcione productos o servicios que puedan perjudicar la moralidad de este público.
•	Cláusula de recogida de información con fines comerciales: esta cláusula será necesaria en el caso de que tu formulario recoja datos del usuario y usted vaya a utilizar esa información para cualquier tipo de fin comercial.
Estas las podremos activar dependiendo del uso de cada uno de nuestros formularios de recogidas de información y dependiendo del tipo de negocio.

= ¿Cómo puedo saber si mis formularios cumplen con la RGPD? =
El plugin le notificará si cuenta con algún formulario que no cumpla con el reglamento.

= ¿Por qué no puedo habilitar las cláusulas sin activar la de tratamiento de información en la sección de formularios? =
Esto es debido a que en TODOS los casos es obligatorio informar al usuario de tu política de privacidad, antes de que este pueda enviar a la empresa cualquier tipo de comunicación electrónica.

= ¿Qué son las cookies? =
Una cookie es una pequeña información enviada por un sitio web y almacenada en el navegador del usuario, de manera que el sitio web puede consultar la actividad previa del usuario.
Sus principales funciones son:
Llevar el control de usuarios: cuando un usuario introduce su nombre de usuario y contraseña, se almacena una galleta para que no tenga que estar introduciéndolas para cada página del servidor. Sin embargo, una galleta no identifica a una persona, sino a una combinación de computadora de la clase de computación-navegador-usuario.
Conseguir información sobre los hábitos de navegación del usuario, e intentos de spyware (programas espía), por parte de agencias de publicidad y otros. Esto puede causar problemas de privacidad y es una de las razones por la que las cookies tienen sus detractores.

= ¿Qué cookies utiliza mi web? =
! NO TE PREOCUPES ¡ El plugin recoge las cookies que actualmente utiliza nuestro sitio web y además especifica al usuario el tipo de cookie que es y para qué sirve, en el caso de que no conozcamos la cookie que utiliza podrás añadir esa información de forma manual, y esta información será añadida a tu página de Política de Cookies.

= ¿Qué pasa si mi web utiliza una nueva cookie no registrada anteriormente? =
El plugin te  avisará, lo único que tendrá que hacer será añadir la información en el caso de que no la tenga y el plugin la añadirá a su Política de cookies. 
Si esta no es reconocida por el plugin le recomendamos que se ponga en contacto con clickDatos.

= ¿Qué puedo hacer para no sobrescribir una página? =
En nuestro plugin añadimos a cada página una etiqueta HTML, deberá eliminar el atributo id=\"clickdatos\" del su página para que el plugin no sobrescriba dicha página al actualizarlas.

= ¿Tengo dudas sobre el RGPD en mi negocio? ¿Qué hago? =
En el menú principal del plugin encontrará mucha información sobre el RGPD. Si aun así, sigue teniendo dudas puede enviarnos un email con su consulta desde el mismo plugin, clickDatos le hará una valoración de su web y se pondrá en contacto con usted lo antes posible.

= ¿Qué es el SSL? ¿Es necesario? =
SSL significa \"Secure Sockets Layer\". Es un protocolo diseñado para proteger la información de ida y vuelta que se transmite desde tu ordenador hasta la web. 
Un certificado SSL es esencial en nuestra página web puesto que su función es la de proteger a los visitantes de la web y sus datos, decirles que el sitio web es auténtico, real y de confianza para ingresar sus datos personales.
